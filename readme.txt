# Reviewing the homework

This document provides step-by-step instructions on how to run my solution for given tasks (Task 1 & Task 2).

## Prerequisites

Before running the solution, make sure you have the following installed on your system:

- Python (version 3.6.8)
- The Python Standard Library

## Installation

1. Clone the repository to your local machine.
2. Navigate to the project directory using the command line.

## Running the Solution

To run the solution, follow these steps:

1. Open a terminal or command prompt.
2. Navigate to the project directory if you haven't already.
3. Execute the following command to run the Python script:

Running a few unit tests (with pytest):
``` pytest datapao_hw.py  ```
a
Generating the output files:
``` python main.py task1 ```
``` python main.py task2 ```
